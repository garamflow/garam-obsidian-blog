# 18-1. 댓글 수정의 개요
- 댓굴 수정 페이지는 모ㄹ 기능을 이용해 구현한다.
- 모달(modal)은 웹 페이지에 새 창을 띄우는 팝업 창과 달리 같은 웹 페이지 내부에서 상위 레이어를 띄우는 방식으로 사용하는 창이다.
	- 모달 창이 뜨면 기존 창은 비활성 상태가 되고, 모달 창을 종료해야만 원래 화면으로 돌아갈 수 있다.
