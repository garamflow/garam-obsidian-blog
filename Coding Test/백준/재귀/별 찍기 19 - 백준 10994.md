# 별 찍기 19 - 백준 10994
## 문제
- 예제를 보고 규칙을 유추한 뒤에 별을 찍어 보세요.
### 입력
- 첫째 줄에 N(1 ≤ N ≤ 100)이 주어진다.
### 출력
- 첫째 줄부터 차례대로 별을 출력한다.
### 예제
```java
// 입력
1

// 출력
*
```

```java
// 입력
2

// 출력
*****
*   *
* * *
*   *
*****
```

```java
// 입력
3

// 출력
*********
*       *
* ***** *
* *   * *
* * * * *
* *   * *
* ***** *
*       *
*********
```
